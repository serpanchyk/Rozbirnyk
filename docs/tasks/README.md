# Tasks

This folder tracks project tasks as individual Markdown files.

Task files are organized by status:

- `done/`
- `backlog/`
- `blocked/`

Each task file should capture:

- goal
- status
- acceptance criteria
- related docs
- notes if needed

Current task folders:

- `done/` contains completed tasks migrated from the former `TASKS.md`
- `backlog/` is empty
- `blocked/` is empty
